# Netum-Summer-2022

Pre-assignment for Software Development summer job 2022 at Netum

Full-stack application to fetch, add, edit and delete persons in a `MySQL` database.<br/>
Backend is made with `TypeScript` & `NodeJS`, frontend with `TypeScript` & `React`.

The app is running at [Heroku](https://netum-summer.herokuapp.com).



## Database

The database is made with `MySQL` add-on in Heroku.

``` sql
CREATE TABLE persons
(
    id          INTEGER         NOT NULL    PRIMARY KEY     AUTO_INCREMENT
    , fname     VARCHAR(100)    NOT NULL
    , lname     VARCHAR(100)    NOT NULL
    , age       INTEGER         NOT NULL
) ENGINE=InnoDB DEFAULT CHARSET=utf8;
```

`id`    - an integer, autogenerated by the database<br/>
`fname` - person's firstname, 100 characters max. length<br/>
`lname` - person's lastname, 100 characters max. length<br/>
`age`   - person's age, an integer

### Using locally:

<p style="color: red">
    NOTE! REMEMBER TO CHANGE YOUR OWN MYSQL CREDENTIALS IN <code>config/db.config.ts</code>!
</p>

* Install with `yarn install` (or if you prefer npm: `npm install`)<br/>

Because `NodeJS` works with `JavaScript`, we need to convert the `TypeScript` files into js files.<br/>

* For this, you can use command that is configured in `package.json` with `yarn build`<br/>
* Now you can start the app with `yarn start`<br/>

Default port is 8080, so you should now be able to access the app with your browser at [http://localhost:8080](http://localhost:8080).
